What is ORAS Artifact Explorer?

ORAS is the de facto tool for working with OCI Artifacts. It treats media types as a critical piece of the puzzle. Container images are never assumed to be the artifact in question. ORAS provides CLI and client libraries to distribute artifacts across OCI-compliant registries. ORAS is the de facto tool for working with OCI Artifacts.
